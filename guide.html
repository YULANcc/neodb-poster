The Shelf 使用指南

NeoDB Personal Collection Poster Exporter

1
设置 NeoDB 连接
点击右上角的齿轮图标，展开设置面板。填写你的 NeoDB 实例地址和 Access Token。
如何获取 Token？
访问 neodb.social → 设置 → 开发者 → 创建应用。Token 会保存在本地，下次无需重复填写。
2
加载收藏数据
点击同步图标加载数据。首次加载每个分类的首页（约 200 条），底部会出现「Load More」按钮，按需逐批加载更多数据。
分页加载：大收藏量用户无需等待全量加载，按需加载更多旧数据
3
筛选与排序
使用顶部的筛选栏选择分类、状态和日期范围。点击任意海报卡片可以查看详细信息。
视图切换：By Month（按月分组）或 Flat（平铺展示）
4
导出海报
点击 Export Poster 按钮，预览生成的海报效果，满意后点击下载 PNG 文件。
海报内容：标题、副标题、封面、统计信息，适合社交分享
5
导出单张卡片
点击任意卡片查看详情，在详情底部点击「Export Card」即可导出单张卡片的完整信息（封面、标题、评分、评语、标签），PNG 格式。
适合分享：单张卡片适合社交媒体、笔记、收藏记录
常见问题
支持哪些 NeoDB 实例？
默认支持 neodb.social，也支持自建实例，在设置中修改 Instance URL 即可。
海报导出失败怎么办？
确保网络连接正常，封面图片可访问。如果问题持续，可以尝试刷新页面后重试。
我的 Token 安全吗？
Token 以编码形式保存在 localStorage，不会上传到任何服务器。所有数据仅在浏览器本地处理。长期建议使用 OAuth PKCE 流程进一步降低风险。
